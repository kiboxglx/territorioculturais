# Território Culturais

Site institucional do **Território Culturais** — plataforma de projetos de economia criativa, cinema e cultura em Minas Gerais.

Built with React 19 + Vite 6 + Tailwind CSS v4.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| UI | React 19 + JSX |
| Build | Vite 6 |
| Styling | Tailwind CSS v4 |
| Routing | React Router v7 |
| Animation | Framer Motion v12 |
| Icons | Lucide React |
| SEO | React Helmet Async |

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Production build
npm run build

# Preview production build locally
npm run preview

# Lint
npm run lint
```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── pages/          # Route-level page components
├── data/           # Static data (projects, assets manifest)
└── index.css       # Global styles + Tailwind entry
public/
├── images/         # Static images (hero, projects, events)
└── events/         # Event media assets
```

## Notes

- Large media files (`.mp4`, `.mov`) are excluded from git — use a CDN or cloud storage.
- Images in `public/` are served as-is; WebP versions are preferred for performance.
- The `dist/` folder is generated by `npm run build` and is gitignored.
